# ControlPanel for DRSSTC (RP2350 + LVGL)

Universal control panel (interapter) for Tesla coil (DRSSTC) based on Raspberry Pi RP2350 microcontroller.


The project is in alpha development. Treat it as something new in the Tesla development. Bruh

📦️ Install:

git clone [text](https://codeberg.org/uki_chan228/ControlPanel_for_DRSSTC)

git submodule update --init --recursive
